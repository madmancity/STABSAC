STATSAC - Statistical Tools Assisting Tournament Simulation, Analysis, and Creation
Created by Liam C. Ray

ACTIVE:
genbracket.py
=================================================================================
Simulates progression of a tournament with editable decision mechanisms,
outputs full bracket of results. Later versions to include data visualization,
custom bracket presets, stat integration, and procedural/step by step generation.
=================================================================================

PLANNED:
buildbracket.py
=================================================================================
Will allow custom bracket creation, blank, seeded, or full. May include image
generation as well.
=================================================================================

TourneyWizard
=================================================================================
Application using machine learning to analyze matchups and predict odds, as well
as analyzing existing tournaments by generating data on teams, correlating wins
to stats, and in depth matchup scenario analysis.
=================================================================================
